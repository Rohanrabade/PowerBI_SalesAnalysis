# PowerBI_SalesAnalysis
# 📊 Power BI Sales Analysis Dashboard

## 📌 Overview
This project analyzes sales data using Power BI to uncover trends, profitability, and customer insights for better decision-making.

---

## 🎯 Objectives
- Analyze **quarterly sales trends**
- Measure **YoY growth**
- Identify **top product categories**
- Find **profitable & loss-making sub-categories**
- Determine **loyal customers**

---

## 📈 Key Insights

- 📊 **Quarterly Trends:** Sales increase from Q1 to Q4, highest in Q4  
- 📉 **YoY Growth:** Peak in 2016, slight dip in 2017  
- 🏆 **Top Category:** Technology leads in both sales and profit  
- 💰 **Most Profitable:** Copiers, Accessories, Phones  
- ❌ **Least Profitable:** Tables, Bookcases, Supplies  
- 👥 **Top Customers:** Sean Miller, Tamara Chand, Raymond Buch  

---

## 🛠️ Tools Used
- Power BI  
- DAX  
- Data Modeling & Visualization  

---

## 📌 Conclusion
Technology drives the most revenue and profit, while some furniture items cause losses. Sales peak in Q4, showing clear seasonal trends.
